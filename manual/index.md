## CLI Usage

```bash
tesl help                    # Show this help
tesl help manual             # Show manual index
tesl help manual overview   # Show overview
tesl help manual examples   # Show examples list
tesl help examples           # Same as above
tesl help search <query>     # Search documentation
```
